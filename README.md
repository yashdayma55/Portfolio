# Yash Dayma — Portfolio

A modern, minimal portfolio built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **Sections**: Hero, Skills, Projects, Achievements, Contact
- **Design**: Purple-pink gradient accents, subtle shadows, clean cards
- **Responsive**: Mobile-first layout
- **Smooth scrolling** for navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Customization

- **Add CV**: Place your CV PDF in `public/cv.pdf` — the "Download CV" button will then link to it
- **Projects**: Edit `src/components/Projects.tsx` to add/update projects
- **Skills**: Edit `src/components/Skills.tsx`
- **Achievements**: Edit `src/components/Achievements.tsx`

## Deploy

```bash
npm run build
npm start
```

Or deploy to [Vercel](https://vercel.com) for zero-config hosting.
